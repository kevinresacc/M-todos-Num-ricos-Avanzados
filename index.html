Métodos Numéricos Avanzados

Herramienta para calcular raíces de funciones con métodos iterativos

Método del Punto Fijo

Ingrese la función g(x) donde x = g(x), el valor inicial, el error mínimo y el máximo de iteraciones.

Funciones matemáticas disponibles:

  • sin(x) - Seno
  • cos(x) - Coseno
  • tan(x) - Tangente
  • log(x) - Logaritmo natural
  • log10(x) - Logaritmo base 10
  • exp(x) - Exponencial e^x
  • sqrt(x) - Raíz cuadrada
  • abs(x) - Valor absoluto
  • π - Pi (3.1416...)
  • e - Número de Euler (2.7182...)
  • x^y - x elevado a la y (potencia)
Ejemplo: x^2 para x², 2^x para 2ˣ, x^(1/2) para √x

Método de Newton-Raphson

Ingrese la función f(x), el valor inicial, el error mínimo y el máximo de iteraciones. La derivada se calculará automáticamente, pero puede editarla si es necesario.

Nota: La derivada se calcula automáticamente. Para funciones complejas, puede editar manualmente la derivada.
